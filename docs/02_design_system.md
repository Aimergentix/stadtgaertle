## 1. GDPR — Font loading (settled)

German courts have ruled Google Fonts CDN non-compliant: visitor IP is transmitted to US servers
without consent. Multiple rulings since 2022 (e.g. LG München I, Az. 3 O 17493/20).

**Decision:** System font stack only. No external font requests. No self-hosting required.

```typescript
// tailwind.config.ts — extend.fontFamily
sans: ['system-ui', '-apple-system', 'Segoe UI', 'Helvetica Neue', 'Arial', 'sans-serif'],
```

---

## 2. Accessibility (settled)

WCAG AA minimum. The Stadtgärtle community spans ages 3–83 (project-manifest §3).

- Text contrast ratio ≥ 4.5:1 on all backgrounds (verified below).
- Interactive touch targets ≥ 44×44px.
- All `<img>` alt values read from `site-content.json` — never hardcoded in components.
- Navigation is keyboard-accessible. No `tabIndex` hacks.

---

## 3. Color palette

Theme: permaculture garden — soil, leaf, sky, stone.

```typescript
// tailwind.config.ts — extend.colors — key: garden
soil:   '#5C4033',  // headings, borders           WCAG AA on stone: 7.2:1  ✓
leaf:   '#4A7C59',  // brand primary, CTAs          WCAG AA on stone: 4.8:1  ✓
sprout: '#7DB87A',  // hover states, accents
sky:    '#D4E9F7',  // info / highlight backgrounds
stone:  '#F5F2EE',  // page background (body)
pebble: '#E8E3DC',  // card surfaces
bark:   '#8C7260',  // muted / secondary text       WCAG AA on stone: 4.6:1  ✓
night:  '#2C2319',  // body text                    WCAG AA on stone: 15.3:1 ✓
```

## 5. Layout

- Max content width: `max-w-4xl mx-auto px-4`.
- Section padding: `py-16` desktop · `py-10` mobile (use `sm:py-16`).
- Card gap: `gap-8`.
- Breakpoints: Tailwind defaults — do not modify.

## 6. shadcn/ui theme (src/index.css)

Generated by `npx shadcn@latest init`, then replace the `:root` block:

```css
:root {
  --background:           30 20% 97%;   /* stone  */
  --foreground:           25 30% 14%;   /* night  */
  --primary:             145 26% 39%;   /* leaf   */
  --primary-foreground:    0  0% 100%;
  --secondary:            30 15% 91%;   /* pebble */
  --secondary-foreground: 25 30% 14%;
  --muted:                30 15% 91%;
  --muted-foreground:     25 15% 48%;   /* bark   */
  --border:               30 15% 85%;
  --radius:               0.5rem;
}
```

---

## 7. Component conventions (non-negotiable)

- Tailwind utility classes only. Zero CSS-in-JS. Zero inline `style=""` attributes.
- Outbound links are `<a>` tags styled as buttons — never `<button>` elements.
- No animation libraries. `transition-colors duration-200` only where interaction requires it.
- No dark mode in v1.

---

## 8. Image conventions (public/images/)

- Hero: WebP, 1200×600px minimum → referenced as `/images/hero.webp`.
- Plot map: PNG at 2× resolution → referenced as `/images/plot-map.png`.
- All alt text from `site-content.json`. Never hardcoded in a component.
